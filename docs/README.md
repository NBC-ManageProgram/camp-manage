# 📚 기능 목록

## 프로그램 시작 화면

- [x] `1` 입력 시 수강생 관리로 이동
- [x] `2` 입력 시 점수 관리로 이동
- [x] `3` 입력 시 프로그램 종료

### 🔥 출력 요구사항

- [x] 프로그램 시작 시 출력

```java
"=================================="
"내일배움캠프 수강생 관리 프로그램 실행 중..."
"1. 수강생 관리"
"2. 점수 관리"
"3. 프로그램 종료"
"관리 항목을 선택하세요..."
```

### 🔥 입력 요구사항

- [x] 1 이상 3이하 정수

```java
1
```

### 🔥 예외 처리 사항

- [x] `[ERROR]`로 시작하는 에러 메시지를 출력한다.
- [x] 에러 메시지를 출력 후, 그 부분부터 입력을 다시 받는다.
- [x] 1 이상 3 이하의 정수가 아닌 경우
- [x] 숫자가 아닌 자료형이 들어올 경우

```java
"[ERROR] 잘못된 값을 입력하였습니다. 다시 입력해주세요."
```

## 수강생 관리

- [x] `1` 입력 시 수강생 등록으로 이동
- [x] `2` 입력 시 수강생 조회로 이동
- [x] `3` 입력 시 수강생 수정으로 이동
- [x] `4` 입력 시 수강생 삭제로 이동
- [x] `5` 입력 시 프로그램 시작 화면으로 이동

### 🔥 출력 요구사항
+ [x] 수강생 관리 메뉴 출력
```java
"=================================="
"1. 수강생 등록"
"2. 수강생 조회"
"3. 수강생 수정"
"4. 수강생 삭제"
"5. 돌아가기"

"관리 항목을 선택하세요..."
```

+ [x] 돌아가기 선택 시 출력
```java
"이전 메뉴로 돌아갑니다."
```

### 🔥 입력 요구 사항

- [x] 1 이상 5이하의 정수

```java
1
```

### 🔥 예외 처리 사항

- [x] `[ERROR]`로 시작하는 에러 메시지를 출력한다.
- [x] 에러 메시지를 출력 후, 그 부분부터 입력을 다시 받는다.
- [x] 1 이상 5 이하의 정수가 아닌 경우
- [x] 숫자가 아닌 자료형이 들어올 경우

```java
"[ERROR] 잘못된 값을 입력하였습니다. 다시 입력해주세요."
```

## 수강생 등록

- [x] 이름과 필수 과목, 선택 과목, 상태를 입력 받아 새로운 수강생을 등록한다.
- [ ] 등록이 완료되면 고유번호를 지정하여 알려준다.

### 🔥 출력 요구 사항

- [x] 이름 입력 시 출력

```java
"=================================="
"[[ 이름을 입력해주세요. ]]"
```

- [x] 필수 과목 입력 시 출력

```java
"=================================="
"[[ 필수 과목을 3개 이상 선택해주세요. ]]"
"((ex) 1 2 3)"
"1. Java"
"2. 객체지향"
"3. Spring"
"4. JPA"
"5. MYSQL"
```

- [x] 선택 과목 입력 시 출력

```java
"=================================="
"[[ 선택 과목을 2개 이상 선택해주세요. ]]"
"((ex) 1 2)"
"1. 디자인 패턴"
"2. Spring Security"
"3. Redis"
"4. MongoDB"
```

- [x] 상태 입력 시 출력

```java
"=================================="
"[[ 상태를 설정해주세요 ]]"
"1. Green"
"2. Yellow"
"3. Red"
```

- [x] 등록 완료 시 출력

```java
"=================================="
"[[ 등록이 완료되었습니다. ]]"
"|  고유번호  |  이름  |  과목  |  상태  |  과목명  |"
"고유번호는 [ ] 번 입니다."
```

### 🔥 입력 요구사항

**이름 입력 요구사항**

- [ ] 공백을 포함하지 않는 문자열로 입력받는다.
- [ ] 한글로만 입력받는다.
- [ ] 2글자 이상 10글자 이하의 범위로 입력받는다.

```java
송선호
```

**필수 과목 입력 요구사항**

- [ ] 1 이상 5이하의 정수를 입력받는다.
- [ ] 숫자가 아닌 자료형이 들어올 경우
- [ ] 정수는 최소 3개 이상 입력받아야 한다.
- [ ] 각 정수는 공백으로 구분한다.

```java
1 3 5
```

**선택 과목 입력 요구 사항**

- [ ] 1 이상 4이하의 정수를 입력받는다.
- [ ] 정수는 최소 2개 이상 입력받아야 한다.
- [ ] 각 정수는 공백으로 구분한다.

```java
3 4
```

**상태 입력 요구 사항**

- [ ] 1 이상 3이하의 정수를 입력받는다.

```java
1
```

### 🔥 예외 처리 사항

- [ ] `[ERROR]`로 시작하는 에러 메시지를 출력한다.
- [ ] 에러 메시지를 출력 후, 그 부분부터 입력을 다시 받는다.

**이름 예외 처리 사항**

- [x] 한글이 아닌 다른 문자열이 들어올 경우
- [x] 공백이 들어올 경우
- [x] 2글자 이상 10글자 이하의 이름이 아닌 경우

```java
"[ERROR] 잘못된 입력입니다. 공백을 제외한 2글자 이상 10글자 이하의 한글로 입력해 주세요."
```

**필수 과목 예외 처리 사항**

- [x] 1 이상 5 이하의 정수가 아닌 경우
- [x] 숫자가 아닌 자료형이 들어올 경우
- [x] 숫자의 개수가 2개 이하일 경우
- [x] 숫자가 중복될 경우
- [x] 공백이 아닌 다른 구분자가 들어올 경우

```java
"[ERROR] 잘못된 입력입니다. 1 이상 5 이하의 정수로 서로 다른 3개 이상의 필수 과목을 선택해주세요."
"각 과목은 공백으로 구분되어야 합니다."
```


**선택 과목 예외 처리 사항**

- [ ] 1 이상 4 이하의 정수가 아닌 경우
- [ ] 숫자가 아닌 자료형이 들어올 경우
- [ ] 숫자의 개수가 1개일 경우
- [ ] 숫자가 중복될 경우
- [ ] 공백이 아닌 다른 구분자가 들어올 경우
```java
"[ERROR] 잘못된 입력입니다. 1 이상 4 이하의 정수로 서로 다른 2개 이상의 필수 과목을 선택해주세요."
    "각 과목은 공백으로 구분되어야 합니다."
```

**상태 예외 처리 사항**

- [ ] 1 이상 3 이하의 정수가 아닌 경우
- [ ] 숫자가 아닌 자료형이 들어올 경우

```java
"[ERROR] 잘못된 입력입니다. 1 이상 3 이하의 정수로 입력해주세요."
```

## 수강생 조회

- [ ] 수강생 조회에 대한 메뉴를 출력받는다.
- [ ] `1` 입력 시 수강생 전체 정보를 조회한다.
- [ ] `2` 입력 시 수강생 상태별 수강생 목록을 조회한다.
- [ ] `3` 입력 시 수강생 관리 화면으로 돌아간다.

### 🔥 출력 요구사항

- [ ] 수강생 조회 메뉴 출력

```java
"=================================="
"1. 수강생 전체 정보 조회"
"2. 수강생 상태별 수강생 목록 조회"
"3. 돌아가기"
```

- [ ] `1`수강생 전체 정보 조회 선택 시 화면 출력

```java
"=================================="
"[[ 수강생 전체 정보  ]]"
"|  고유번호  |  이름  |  상태  |    과목명     |"
"|  고유번호  |  이름  |  상태  |  java, spring,.. |"

```

- [ ] `2`수강생 상태별 수강생 목록 조회 선택 시 화면 출력

```java
"=================================="
"[[ 수강생 상태별 수강생 정보 ]]"
"[ Green ]"
"|  고유번호  |  이름  |"

"[ Yellow ]"
"|  고유번호  |  이름  |"

"[ Red ]"
"|  고유번호  |  이름  |"
```

+ [ ] 돌아가기 선택 시 출력
```java
"이전 메뉴로 돌아갑니다."
```

### 🔥 입력 요구사항

- [ ] 1 이상 3 이하의 정수

```java
1
```

### 🔥 예외 처리 사항

- [ ] `[ERROR]`로 시작하는 에러 메시지를 출력한다.
- [ ] 에러 메시지를 출력 후, 그 부분부터 입력을 다시 받는다.
- [ ] 1 이상 3 이하의 정수가 아닌 경우

```java
"[ERROR] 잘못된 입력입니다. 1 이상 3 이하의 정수로 입력해주세요."
```

```java
"2. 상태 정보 수정"
```

```java

"[[ 수정할 이름을 입력해주세요 ]]"
```

- [ ] `2`상태 수정 시 출력

```java

| 고유번호 | 이름 | 상태 |

"3. Red"
```

- [ ] 삭제 여부 입력에서 `1` 입력 시 수강생 삭제를 진행한다.
- [ ] 삭제 여부 입력에서 `2` 입력 시 수강생 관리 메뉴로 돌아간다.

```java
"=================================="

```

- [ ] 삭제 확인 여부 출력

```java
"[[ 삭제하시겠습니까? ]]"
"|  고유번호  |  이름  |  상태  |    과목명    |"
"1. 삭제 2. 돌아가기"
```

- [ ] `1`삭제 완료 출력

```java
"삭제가 완료되었습니다"
```
+ [ ] 돌아가기 선택 시 출력
```java
"이전 메뉴로 돌아갑니다."
```

### 🔥 입력 요구 사항

- [ ] 생성된 고유번호 범위 내 정수 입력

```java
1
```

- [ ] 삭제 여부 입력 (1 OR 2)

```java
1
```

### 🔥 예외 처리 사항

- [ ] `[ERROR]`로 시작하는 에러 메시지를 출력한다.


**삭제 확인 여부 시 예외 처리 사항**

- [ ] 1 또는 2가 아닐 경우
